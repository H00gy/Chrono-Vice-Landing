# Chrono-Vice-Landing
## What is Chrono-Vice?
Chrono-Vice is my upcoming 2D wristwatch dealing game centered around buying and selling watches while navigating a trend based simulated economic system. This Website demonstrate gameplay prototypes for Chrono-Vice.

## Inventory System
The inventory system in "Chrono-Vice" will be game object based, showcased in this [prototype](https://github.com/H00gy/inventorySystem).
## Book system
This is a book [prototype](https://github.com/H00gy/Unity-2D-book/blob/main/README.md) that will be used as a reference book in "Chrono-Vice"
